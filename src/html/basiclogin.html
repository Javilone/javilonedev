Basic Login Medac fue un prototipo temprano de login de usuario para un simulador de clínica de higiene bucodental. Dicho simulador, en desarrollo aún por diferentes promociones de alumnos en Fundación MEDAC.

Trabajando de forma colaborativa con múltiples equipos involucrados en el desarrollo, haciendo uso de herramientas de control de versiones como GitHub y Plastic (Unity).
Realizado en Unity, hacía uso de Web Services (EC2 y RDS) para alojar la base de datos SQL. Empleando PHP y C# para la programación .

Hecho con ❤️ por Javier López Lara.
